# 网页小游戏

## 扫雷（mine-sweeper）

[源代码](../../../mine-sweeper) | [体验地址](https://itabbot.com/mine-sweeper/src/index.html)

这是一款由 JavaScript + HTML + CSS 制作的经典扫雷游戏。借助 `window.requestAnimationFrame()` 的能力运行游戏循环；基于 Web Canves API 封装了游戏的核心逻辑；结合特定的 Web Event 实现了与玩家的互动。

其中还加入了一个有趣的动画效果，点击空白格子后，会以骨牌倒下的样式连锁翻开，赶紧来体验一下吧。

## 俄罗斯方块（tetris）

[源代码](../../../tetris) | [体验地址](https://itabbot.com/tetris/src/index.html)

这是一款经典的俄罗斯方块网页小游戏，由 JavaScript + HTML + CSS 制作而成。同样是借助 `window.requestAnimationFrame()` 的能力运行游戏循环；基于 Web Canves API 封装了游戏的核心逻辑；结合特定的 Web Event 实现了与玩家的互动。

俄罗斯方块的变形逻辑非常有趣，它是围绕一个特殊的中心点旋转，而且还无视障碍物，赶紧来体验一下吧。
